## 🧠 System Overview

HelixCortex AI is a **cloud-native, multi-agent AI platform** engineered for complex clinical decision support. Unlike traditional health-trackers, HelixCortex implements a decoupled cognitive architecture, allowing specialized AI agents to operate independently across varied data domains—from real-time patient metrics to deep biomedical research.

---

## 🏗️ Core Architecture: The Hybrid Lakehouse
The system bridges the gap between high-speed clinical intake and long-term medical research through a dual-layer data strategy:

OLTP Layer (PostgreSQL): Facilitates sub-second latency for active patient state management and real-time monitoring.

OLAP Layer (Apache Iceberg): Acts as an immutable, versioned clinical data lake. This allows for "Time Travel" audits and provides the high-fidelity historical data required for training longitudinal ML models.

---

## 🤖 Multi-Agent Orchestration
We utilize a Modular Intelligence Framework where specialized agents collaborate to form a comprehensive clinical picture:

The Data Agent: Synthesizes current state from Postgres with historical context from Iceberg.

The ML Risk Agent: Executes time-series forecasting to predict symptom flare-ups with a probabilistic confidence score.

The Biomedical RAG Agent: Performs semantic retrieval across peer-reviewed clinical corpora (e.g., PubMed) to ground AI logic in established medical science.

The Reasoning Agent: Orchestrates the final clinical rationale, ensuring recommendations are both data-driven and evidence-based.

Design Philosophy: HelixCortex is built on the principle of Clinical Auditability. Every recommendation generated by the system is mapped back to specific patient data points and cited biomedical literature.

---

## 🚀 Implementation Highlights
Asynchronous Processing: High-frequency data ingestion is decoupled from the API layer via an event-driven architecture.

Schema Evolution: Leveraging Apache Iceberg to handle evolving clinical data structures without breaking downstream ML pipelines.

Cloud-Native Scale: Fully containerized with Docker and optimized for serverless deployment via Google Cloud Run.

## 🏗️ System Architecture

HelixCortex AI implements a **Hybrid Lakehouse + Agentic Architecture** to bridge real-time clinical intelligence with long-term biomedical research.

### 🧩 Data Architecture

- **OLTP Layer (PostgreSQL)**  
  Handles high-frequency, low-latency operations such as:
  - patient state updates  
  - session data  
  - real-time metrics  

- **OLAP Layer (Apache Iceberg)**  
  Provides immutable, versioned clinical history:
  - longitudinal patient tracking  
  - ML training datasets  
  - audit-ready data pipelines  

---

### 🧠 Intelligence Layer (Multi-Agent System)

- **Data Agent**  
  Interfaces with PostgreSQL + Iceberg for unified context  

- **Biomedical RAG Agent**  
  Performs semantic retrieval across clinical corpora (PubMed-grade)  

- **ML Prediction Agent**  
  Computes flare-up probability and risk scores  

- **Reasoning Agent (Gemini)**  
  Synthesizes multi-source signals into clinical insights
